Snakes

- онлайн игра про движущееся шарики.

Подключения игрока к игроку осуществляется через протокол MQTT (Message Queuing Telemetry Transport). Это легковесный протокол обмена сообщениями по принципу «издатель-подписчик». Он работает поверх TCP/IP (или других протоколов) через центральный сервер, называемый брокером, который пересылает сообщения от Издателей к Подписчикам, делая связь эффективной и надежной, даже при нестабильных соединениях.

paho.mqtt.client - это Python-библиотека, являющаяся частью проекта Eclipse Paho, которая позволяет приложениям на Python легко подключаться и взаимодействовать с MQTT-брокерами для обмена сообщениями. Она используется в игре Snakes.

pygame - это популярная Python-библиотека, представляющая собой набор модулей для создания 2D-игр и мультимедийных приложений.
